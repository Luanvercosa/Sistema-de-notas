# Sistema-de-notas 


<h1>Contruir um sistema de notas que inclui: </h1>

 <ul>
  <li>Adicionar Aluno;</li>
  <li>Adicionar nota do Aluno;</li>
  <li>Consultar a nota do aluno;</li>
  <li>Calcular a média do aluno;</li>
  <li>Listar alunos com a maior nota;</li>
  <li>Remover aluno.</li>
</ul>
